Resources

This website gives a little bit history about Coffee in general. Also, it introduces Speciality coffee and how it is different than regular coffee.

I think I did pretty well in designing how to make perfect coffee page. I have included several images and collected a good amount of information about the equipment that are used to make a coffee.

My project needs a little more information to be added in order to cover more aspects about coffee.

For the javascript, I used it to enable the user to navigate the website through the navigation menu.

I did some changes to the first image in the History webpage using Photoshop.

I have enabled the user to print any page of my website. I did that using print.css file inside the css folder.
